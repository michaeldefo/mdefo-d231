# Utilisation de *GitHub Pages*** — pour créer un site statique D231

***

## 🌐 GitHub Pages — C’est quoi ?

**GitHub Pages** est un service gratuit de GitHub permettant d’héberger directement un **site web statique** (HTML, CSS, JS) depuis un dépôt GitHub.

Tu peux l’utiliser pour :

*   un site vitrine,
*   un portfolio,
*   une documentation,
*   un blog statique (ex : Jekyll),
*   des prototypes web,
*   ou même l’hébergement d’une webapp front-end (React, Vue via build statique).

***

## 🚀 Etapes du processus

### **1. Crée un dépôt GitHub vide on va y pousser nos fichiers**

Nom au choix, par exemple `mdefo-d231`.

### **2. Active GitHub Pages**

Dans ton dépôt :

1.  Va dans **Settings**
2.  Dans la barre latérale : **Pages**
3.  Section *Build and deployment* :
    *   **Source** : `Deploy from a branch`
    *   **Branch** : `main` + dossier `/ (root)`
4.  Clique **Save**

### **4. Obtiens ton URL**

Tu obtiens alors automatiquement :

    https://<ton-nom-github>.github.io/<nom-du-repo>/

Exemple :

    https://mdefo-d231.github.io/mdefo-d231/
